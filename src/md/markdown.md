# 마크다운 문법

## 0. 정의

Markdown 문법은 복잡한 태그 없이 간단하고 사람이 읽기 쉬운 구문을 사용하여 서식 요소를 추가하는 데 사용할 수 있는 경량 마크업 언어이다. HTML 경험이 없는 사람도 쉽게 읽고 쓰고 HTML으로 변환이 가능하다. md라는 확장자를 사용한다.

## 1. 역사

2004년 John Gruber가 Aaron Swartz와 협력하여 마크다운을 만들었다. 2002년 출시된 reStructedText 언어에서 영감을 얻었으며 기존에는 HTML로 렌더링 되지 않고 마크업이 어떻게 보일지 시각적인 암시만 제공했지만 Markdown은 서식이 적용된 출력을 생성할 수 있게 되었다. 기존 Markdown을 보완하기 위한 구문이 있다.

## 2. 장점

1. 문법이 간단하고 직관적이다.
2. Github, Notion, Confluence, Slack 등 마크다운을 지원하는 에디터를 사용할 수 있는 환경에서 추가 학습 없이 가독성 좋은 글을 작성할 수 있다.
3. 일반 텍스트 파일로 용량이 적다.
4. Git과 같은 버전 관리 시스템을 사용해 변경 사항을 관리하기 좋다.

## 3. 구문

### 3-1. 제목

6개의 단계로 헤더(제목)을 지정 가능하며 `<h1>, <h2>, <h3>, <h4>, <h5>, <h6>`태그로 변환된다.
하나의 마크다운 파일에서는 하나의 H1 (대제목) 만 사용해야 한다.

문법<br>

```
# Header 1
## Header 2
### Header 3
#### Header 4
##### Header 5
###### Header 6
```

결과

# Header 1<br>

## Header 2<br>

### Header 3<br>

#### Header 4<br>

##### Header 5<br>

###### Header 6<br>

<br>
<br>
h1, h2는 =, - 기호로도 사용 가능하다.<br>
Header 1<br>
=<br>
Header 2<br>
-<br>

### 3-2. 글자 강조

글자의 기울임(`<em>`), 두꺼움(`<strong>`), 취소선(`<del>`)을 변경할 수 있다.
밑줄의 경우 마크다운에서 지원하지 않아 `<u>밑줄</u>`를 직접 입력해서 변경한다.<br>

문법<br>

```
*기울여쓰기* / _기울여 쓰기_
**굵게 쓰기** / __굵게 쓰기__<br>
~~취소선~~
```

<br>

결과
_기울여쓰기_ / _기울여 쓰기_<br>
**굵게 쓰기** / **굵게 쓰기**<br>
~~취소선~~<br>

### 3-3. 목록

`<ol>, <ul>, <li>` 태그로 변환되는 목록을 표현한다.
`1.` 로 시작하는 항목을 작성하면 숫자가 있는 `<ol>`로 변환되며 `-, *, +`로 시작하는 항목은 점 목록`<ul>`로 변환된다.
숫자와 기호는 섞어서 사용 가능하다.

1. 첫번째 항목
1. 첫번째 항목

- 두번째 항목
  - 두번째 항목

### 3-4. 인용문

`>` 일반 인용문, `>>` 이중 인용문으로 이용 가능하다.

구문<br>

```
> 인용문
>> 이중 인용문
```

<br>

결과<br>

> 인용문<br>
>
> > 이중 인용문<br>

### 3-5. 인라인 코드/코드 블럭

강조할 코드를 인라인 코드는 \` 기호 1개, 코드 블럭은 \` 기호 3개 이상으로 감싸 표현한다.
백틱(\`) 기호를 출력하기 위해서는 `\` 를 기호 앞에 입력한다.

문법

````
`인라인 코드`

\```
코드블럭
\```

\`백틱
````

결과

`인라인 코드`

```
코드블럭
```

\`백틱

### 3-6. 수평선 넣기

-, \*, \_ 기호를 3개 이상 작성해 페이지 구분을 위해 수평선을 넣을 수 있다.

문법

```
---
***
___
```

결과

---

---

---

### 3-7. 표

`<table>` 태그로 변환되는 표를 표현한다. 표 안에는 폰트 스타일을 적용할 수 있다.
`|` 기호로 셀을 구분해서 작성하며, 하이픈(-) 과 세미콜론(:)을 이용해 오른쪽 왼쪽, 가운데 정렬이 가능하다.

문법

```
|왼쪽|가운데|오른쪽|
|---|:---:|---:|
|셀왼쪽정렬|셀중간정렬|셀오른쪽정렬|
```

결과

| 왼쪽       |   가운데   |       오른쪽 |
| ---------- | :--------: | -----------: |
| 셀왼쪽정렬 | 셀중간정렬 | 셀오른쪽정렬 |

### 3-8. 주석

`<!-- -->`, `[//]: #` 기호를 이용해 주석을 표현한다.

문법

```
<!-- 주석 1번 -->
[//]: # 주석 2번
```

### 3-9. 줄바꿈

줄바꿈을 위해서는 문장 마지막에서 띄어쓰기를 2번 하거나 <br> 태그를 직접 입력해서 이용할 수 있다.

문법

```
첫번째줄
두번째줄<br>
세번째줄<br><br>
```

결과  
첫번째줄  
두번째줄<br>
세번째줄<br><br>

### 3-10. 링크

`<a>`로 변한되는 링크를 표현하며 인라인 링크, url 링크, 참조 링크 기능을 활용할 수 있다.

문법

```
[Google](https://google.com)
[Google2](https://google.com "구글링크입니다.")
```

결과  
[Google](https://google.com)  
[Google2](https://google.com '구글링크입니다.')

### 3-11. 이미지

<img>로 변환되는 이미지로 링크와 비슷한 문법을 가지고 있으며 앞에 `!`를 추가해야 한다.

문법

```
![이미지1](https://upload.wikimedia.org/wikipedia/commons/b/b8/PhodopusSungorus_2.jpg)
```

결과
![이미지](https://upload.wikimedia.org/wikipedia/commons/b/b8/PhodopusSungorus_2.jpg)
